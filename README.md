# On My Way to Work

## Overview

**On My Way to Work** is a 2D platformer inspired by classic games such as *Mario*, set in a stylized version of New York City.

The player takes control of an office worker who must reach the workplace as quickly as possible while navigating a dynamic and chaotic urban environment. Progression relies on platforming mechanics that require synchronization with the movement of the city itself.

## Gameplay

The game's core concept is transforming the flow of the city into interactive gameplay elements:

* 🚗 Cars act as platforms.
* 🐦 Pigeons serve as aerial enemies that challenge the player throughout the journey.
* ⚡ At the beginning of each new level, the player can choose a special ability that changes the way the game is played.

These mechanics create a unique urban platforming experience focused on rhythm, adaptation, and quick decision-making.

## Features

* 2D side-scrolling platformer gameplay
* Stylized New York City setting
* Dynamic obstacles and moving platforms
* Multiple special abilities to choose from
* Progressive difficulty across levels
* Urban-themed enemies and hazards

## Technologies

* Python
* Pygame
* PPlay

## Academic Project

This game was developed as a university project, exploring game design concepts, object-oriented programming, and interactive systems through the development of a complete 2D platforming experience.
